About

Welcome! upfast.tf is a web platform that hosts free public Team Fortress 2 servers for anyone to play on. Feel free to join and play if you're interested.

It is a personal project of mine. Servers are hosted on AWS EC2 instances created with Terraform.

Future Plans

• Use Prometheus to monitor server stats & create a status page
• Track stats for total connections to each server
• Allow anyone to create their own server

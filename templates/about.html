About

Welcome! upfast.tf is a web platform that hosts free public Team Fortress 2 servers for anyone to play on. The servers mainly are surf or jump maps so they can be easily played solo. Feel free to join and play if you're interested.

This a personal project of mine. I created it mainly to gain experience provisioning and managing cloud servers.

Servers are hosted on AWS EC2 instances created with Terraform. I monitor the server health with Prometheus.

Future Plans

•  Add more public servers
•  Allow anyone to create and reserve a private server for a set time

Contact

Privacy

upfast.tf doesn't store cookies or use tracking. To get server usage stats, hashed Steam IDs of connected users are recorded.
